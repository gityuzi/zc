flex 伸缩布局盒模型学习

1. 基础

1.1 主轴和侧轴

主轴：可以理解为X轴，和justify-content相关，不一定水平，若其值为column，则方向为垂直。
侧轴：Y轴，方向取决于主轴。

1.2 display

flex：块级元素
inline-flex：行内元素
注意：若display为inline-flex且是浮动或绝对定位，则display为flex。

1.3 flex-container

**flex-direction**：默认row 向右，row-reverse向左；column向下，column-reverse向上。
**flex-wrap**：默认nowrap 单行显示。wrap：多行显示；wrap-reverse：相反。
**flex-flow**：flex-direction + flex-wrap
**justify-content**：默认flex-start：起始位置对齐；flex-end：结束位置对齐；center：中间位置对齐；space-between：均分空隙，首尾顶格。space-around：均分间隙，首尾间隙为中间间隙的一半。
**align-content**:和justify-content差不多，这个是调整行对齐方式。 flex-start | flex-end | center | space-between | space-around | stretch：均分铺满;
**align-items**： flex-start | flex-end | center | baseline | stretch：默认，拉伸项目，填充满整个项目。

1.4 flex-items

**align-self**: auto | flex-start | flex-end | center | baseline | stretch;
**flex-grow**：数值，宽度占container下总和的几分之几。
**flex-shrink**: <number>; /* default 1
**flex-basis**: <length> | auto; /* default auto 如果设置为“0”，不考虑剩余空白空间。如果设置为自动，则按照flex-grow值分配剩余空白空间
**flex**: none | [ <'flex-grow'> <'flex-shrink'>? || <'flex-basis'> ]

应用

1. 居中

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>center</title>
<link rel="stylesheet" href="./center.css">
</head>
<body>
<div class="parent"><div class="child"></div></div>
</body>
</html>

body{
padding: 0;
margin: 0;
}

.parent {
  display: flex;
  height: 300px;
  background-color: black;
}

.child {
  width: 100px;  
  height: 100px;
  margin: auto;  
  background-color: white;
}
