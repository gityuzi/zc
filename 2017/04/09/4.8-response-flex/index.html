百度前端技术学院，小薇学院任务之响应式布局学习。

响应式布局

实现

1. HTML- -meta标签

<meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1, user-scalable=no">

2. CSS-media query

@media screen and (max-width: 980px) {
   head { … }
  #content { … }
  #footer { … }
}

也可以，两个条件限制。

@media only screen and (min-width: 768px) and (max-width: 1024px) {}

3. 还有link标签引用，@importan，xml等方法。

注意

  1. 写HTML的时候class类不同情况下不同。参考bootstrap，但是可能一般的网站并不需要这么写。
  2. 其实感觉还行，不是很难理解。
  3. 使用百分比单位。
